# Changelog

Todos los cambios notables de este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-09-04

### Agregado
- ⏱️ Temporizador de alta precisión para cubo de Rubik
- 🔀 Generador de scrambles aleatorios
- 📊 Sistema de registro y almacenamiento de tiempos
- 📚 Biblioteca de algoritmos OLL (Orientation of the Last Layer)
- 📚 Biblioteca de algoritmos PLL (Permutation of the Last Layer)
- 🎨 Interfaz de usuario moderna con Material Design
- 📱 Navegación con PageView y BottomNavigationBar
- 🗂️ Menú lateral para navegación adicional
- 💾 Persistencia de datos con SharedPreferences
- 🏗️ Arquitectura con Provider para gestión de estado
- 🎨 Diagramas visuales para algoritmos
- 📱 Soporte multiplataforma (Android, iOS, Web, Windows, macOS, Linux)

### Características técnicas
- Flutter SDK 3.3.0+ compatibilidad
- Provider pattern para gestión de estado
- Persistencia local de datos
- Arquitectura modular y escalable
- Interfaz responsive y adaptativa

### Archivos incluidos
- Temporizador principal con funcionalidad completa
- Sistema de scrambles automático
- Visualización de registros históricos
- Biblioteca completa de algoritmos OLL/PLL
- Documentación completa del proyecto
